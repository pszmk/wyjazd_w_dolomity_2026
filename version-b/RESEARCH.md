# Research — Garda ↔ Iseo corridor

Distance and activity research for [Version B](PLAN.md). Road km from route planners and official tourism sources (June 2026). Re-check on [Google Maps](https://maps.google.com) before booking.

**Geographic scope:** the belt from **Lago d’Iseo** (east) through **Valle dei Laghi** and **Valle del Sarca** to **Riva del Garda** (north-west Garda). This is the natural **A4 / SS240 / SS45bis** approach corridor toward the Dolomites — used as staging in the [Version C plan](../version-c/RESEARCH.md) but rarely explored for its own hikes.

## 1. Driving legs (all ≤100 km)

| Leg | km | Time (est.) | Source / route |
|-----|-----|-------------|----------------|
| BGY → Iseo (Camping del Sole) | **~26** | ~35 min | [CamplinQ — 26 km to BGY](https://camplinq.com/camps/camping-del-sole-village/) |
| Iseo → Riva del Garda | **~99** | ~1 h 35 min | [TravelProfy / Rome2Rio](https://www.travelprofy.com/bergamo/riva-del-garda) via A4 Rovato → Brescia → Salò → Riva |
| Riva → Arco | **~8** | ~15 min | SS240 / SS45bis along Sarca valley |
| Arco → Sarche (Toblino) | **~28** | ~40 min | SS45bis north via Dro — [Valle dei Laghi cycle path ref ~27 km Torbole–Toblino](https://www.lifeintravel.it/en/travels/italy/valle-dei-laghi-cycle-path-from-lake-garda-to-lake-toblino.html) |
| Sarche → Iseo | **~99** | ~1 h 40 min | Reverse of Iseo → Riva |
| Sarche → Bressanone (join Dolomites) | **~98** | ~1 h 12 min | [Version C RESEARCH](../version-c/RESEARCH.md) |
| Bressanone → Colfosco | **~55–65** | ~50–60 min | [Skiresort.info](https://www.skiresort.info/ski-resort/alta-badia/arrival-car/) |

**5-day loop total:** ~**262 km** driving (excluding trailhead hops).

**Scenic approach to Version A:** Sarche → Bressanone → Colfosco = **~155 km** over 2 days (same as [Version C days 4–5](../version-c/PLAN.md)) vs Version A’s single **~125 km** Iseo → Colfosco hop.

## 3. Hikes & via ferrata

### Via Ferrata Colodri (Arco) — main ferrata day

| | |
|---|---|
| **Grade** | Easy / A–B — classic **first ferrata** in Garda Trentino |
| **Duration** | ~1 h on wire; 3–4 h round trip with approach |
| **Start** | Parking at Camping Arco Prabi / Via Legionari Cecoslovacchi, Prabi |
| **View** | Sarca valley, Riva, full north Garda |
| **Gear** | Helmet, harness, lanyard — rent in Arco ([Mmove](https://360gardalife.com/en/activities/tours-excursions/viaferrata/colodri-lake-garda-alpine-guide-mmove/), ~€16/day kit) |
| **Sources** | [Visit Trentino](https://www.visittrentino.info/en/guide/tour/via-ferrata-colodri-colt_tour_8279464) · [SummitPost](https://www.summitpost.org/via-ferrata-del-colodri-sentiero-attrezzato-del-colodri/300510) |

**Weather backup:** **Via Ferrata Rio Sallagoni** — short canyon ferrata near Arco ([360gardalife](https://360gardalife.com/en/activities/tours-excursions/viaferrata/)).

### Busatte–Tempesta (Riva / Torbole)

| | |
|---|---|
| **Type** | Panoramic lake path — **400 iron steps**, not a ferrata |
| **Distance** | ~9–10 km loop, ~3 h |
| **Start** | Busatte Adventure Park, above Torbole (~15 min from Riva) |
| **Difficulty** | Medium-easy — vertigo-sensitive people may struggle on steps |
| **Sources** | [Trentino.com](https://www.trentino.com/en/leisure-activities/mountains-and-hiking/hiking-in-spring/busatte-tempesta-panorama-path/) · [Visit Trentino](https://www.visittrentino.info/en/guide/tour/sentiero-busatte-tempesta_tour_1553752) |

### Punta Larici (near Riva)

Short **3.2 km** viewpoint over Garda — [Garda Trentino](https://www.gardatrentino.it/en/activity/covelo-bait-del-germano_45307) lists ~1 h 45 min, +367 m. Good **arrival-day** option if energy allows.

## 4. Climate & timing

- **Sub-Mediterranean** on Garda (olives, lemons) vs **Alpine** at Toblino — pack layers.
- **September** is ideal: warm lake swimming, fewer crowds than August ([Garda Trentino hiking](https://brionegreenresort.com/en/top-trekking-itineraries/)).
- **Busatte steps:** hot metal in afternoon sun — start **early** ([360gardalife](https://360gardalife.com/en/route/torbole-sul-garda-nago-en/walking/lake-garda-panoramic-path-busatte-tempesta/)).
- **Colodri:** sunny east face — avoid midday thunderstorms (same rule as Version A).
