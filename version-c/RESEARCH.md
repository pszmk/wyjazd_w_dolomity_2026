# Research — Budget Garda Loop

Distance and activity research for [Budget Garda Loop](PLAN.md). Road km from route planners and official tourism sources (June 2026). Re-check on [Google Maps](https://maps.google.com) before booking.

**Geographic scope:** **Lago d’Iseo** → **west Garda (Manerba / Salò)** → **Riva del Garda** → **Valle dei Laghi**. The Manerba stop splits the classic Iseo↔Riva transfer and adds Mediterranean west-shore character.

**Plan dates:** **1 – 7 Sep 2026** · **Three camps:** [Camping del Sole](https://www.campingdelsole.it/en/) (Iseo) + [Camping La Rocca](https://www.laroccacamp.it/) (Manerba) + [Camping Brione](https://www.campingbrione.com/EN/) (Riva).

| Riva → Toblino / Sarche (day trip) | **~30–35** RT | ~40 min each way | SS45bis north via Dro |

**Iseo → Riva split:** ~55 km (Day 3) + ~50 km (Day 4) = ~105 km total vs ~99 km direct — similar distance, easier pacing and a new region (west Garda).

## 3. Hikes (main plan)

### Monte Guglielmo (Iseo — Day 2)

| | |
|---|---|
| **Type** | Full-day summit hike above Lake Iseo |
| **Distance** | ~12–14 km, +1,100 m, 5–6 h |
| **Start** | Pescegallo / Castro trailhead (**~10 km** from del Sole) |
| **Difficulty** | Hard — long ascent, exposed upper ridge |
| **Fallback** | Corna Trentapassi (~6–8 km, +500 m) or flat Iseo→Sulzano promenade |
| **Sources** | Local trail maps · [Visit Lake Iseo](https://www.visitlakeiseo.info/en/) |

### Rocca di Manerba (Day 3 optional or Day 4 — from La Rocca camp)

Full optional day plan: **[OPTIONAL-la-rocca-hiking-day.md](OPTIONAL-la-rocca-hiking-day.md)** (hike Day 3 PM, skip Salò, transfer-only Day 4).

| | |
|---|---|
| **Type** | Nature reserve loop — cliff viewpoints over lower Garda |
| **Distance** | ~4–8 km, +200–350 m, 2–3 h |
| **Start** | On foot from Camping La Rocca / Parco Archeologico |
| **Difficulty** | Easy–medium — some exposed sections near cliffs |
| **Bonus** | **Isola San Biagio** (Rabbit Island) — wade or short boat when water level allows |
| **Sources** | [Visit Manerba](https://visitmanerba.it/) · [Garda Outdoors](https://www.gardaoutdoors.com/en/sports/trekking/manerba/) |

### Busatte–Tempesta (Riva / Torbole — Day 5)

| | |
|---|---|
| **Type** | Panoramic lake path — **400 iron steps**, not a ferrata |
| **Distance** | ~9–10 km loop, ~3 h |
| **Start** | Busatte Adventure Park, above Torbole (~15 min from Brione) |
| **Difficulty** | Medium-easy — vertigo-sensitive people may struggle on steps |
| **Sources** | [Trentino.com](https://www.trentino.com/en/leisure-activities/mountains-and-hiking/hiking-in-spring/busatte-tempesta-panorama-path/) · [Visit Trentino](https://www.visittrentino.info/en/guide/tour/sentiero-busatte-tempesta_tour_1553752) |

### Via Ferrata Colodri (Arco — Day 5, main plan)

| | |
|---|---|
| **Grade** | Easy / A–B — classic **first ferrata** in Garda Trentino |
| **Duration** | ~1 h on wire; 3–4 h round trip with approach |
| **Start** | Parking at Camping Arco Prabi / Via Legionari Cecoslovacchi, Prabi (**~8 km** from Brione) |
| **View** | Sarca valley, Riva, full north Garda — **300 m above the lake** |
| **Gear** | Helmet, harness, lanyard — rent in Arco ([Mmove](https://360gardalife.com/en/activities/tours-excursions/viaferrata/colodri-lake-garda-alpine-guide-mmove/), ~€16/day kit) |
| **Sources** | [Visit Trentino](https://www.visittrentino.info/en/guide/tour/via-ferrata-colodri-colt_tour_8279464) · [SummitPost](https://www.summitpost.org/via-ferrata-del-colodri-sentiero-attrezzato-del-colodri/300510) |

### Punta Larici (near Riva — Day 5 backup)

Short **3.2 km** viewpoint over Garda — ~1 h 45 min, +367 m. For non-climbers while others do Colodri.

### Valle dei Laghi — day trip from Brione (Day 6)

| Activity | Distance | Notes |
|----------|----------|-------|
| Castel Toblino lakeshore loop | ~5–8 km | Flat; castle on peninsula |
| Seven Lakes route stage I | ~14 km | Sarche → Monte Terlago — [Outdooractive](https://www.outdooractive.com/en/route/pilgrim-walk/garda-trentino/seven-lakes-route-stage-i-sarche-monte-terlago/312259126/) |
| Covelo → Bait del Germano | ~14 km | View of **four lakes** incl. Garda — [Garda Trentino](https://www.gardatrentino.it/en/activity/covelo-bait-del-germano_45307) |

### Lake Iseo — lakeshore (Day 1)

Flat **Iseo old town** walk — no ferry required. **Monte Isola ferry** from Sulzano (~€9–11 RT) as Day 2 weather backup — [Navigazione Lago d’Iseo](https://www.navigazione.lake-iseo.com/).

## 4. Weather backups

| If… | Then… |
|-----|-------|
| Monte Guglielmo: rain/storm | Corna Trentapassi or Sulzano promenade + Monte Isola ferry |
| Rocca di Manerba: rain | Skip hike; drive straight to Riva; longer waterfront walk |
| Colodri: wet / thunder | **Via Ferrata Rio Sallagoni** (canyon) or **Grotta Cascata Varone** (€7) |
| Busatte: extreme heat | Start at 07:00 or swap to Punta Larici |

**Rio Sallagoni:** [360gardalife](https://360gardalife.com/en/activities/tours-excursions/viaferrata/) · **Varone cave:** [cascata-varone.com](https://www.cascata-varone.com/en/)

## 5. Climate & timing

- **Alpine lake** at Iseo (~200 m) vs **Sub-Mediterranean** west Garda (olives, lemons at Manerba/Salò) vs **Alpine** at Toblino — pack layers.
- **September** is ideal: warm lake swimming, fewer crowds than August.
- **Days 3–4 (west Garda):** hottest part of the trip — swim in afternoon; hike Rocca early.
- **Busatte steps:** hot metal in afternoon sun — start **before 09:00**.
- **Colodri:** sunny east face — avoid midday thunderstorms.
