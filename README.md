- **Car:** Rent at Bergamo airport — essential for camping + trailheads.

- **Via ferrata:** Gran Cir (1 Sep), Ra Gusela (2 Sep) — grade A, beginner-friendly.
- **Book now:** Both campsites, Tre Cime parking or shuttle for **3 Sep**.
